# Playwright Automation Framework – SauceDemo

## 📌 Overview
This project demonstrates an end-to-end UI automation framework built using **Playwright + TypeScript**.

The framework follows best practices like:
- Page Object Model (POM)
- Locator separation
- Test data management
- Config-driven URLs
- Visual testing

---

**--Features Implemented**

1. Login Functionality
2. Inventory Page Validation
3. Visual Testing using Playwright snapshots
